# IndoArt – Collaborative E-Commerce Platform for Indian Artisans

## Overview
IndoArt is a collaborative e-commerce platform designed to empower Indian artisans by enabling them to showcase and sell handcrafted products globally. The platform allows direct communication between customers and artisans to support customization and build trust.

## Objectives
- Promote Indian handicrafts globally
- Eliminate middlemen and increase artisan profits
- Enable real-time customer–artisan interaction
- Preserve India’s cultural heritage using technology

## Tech Stack
Frontend: React.js, HTML, CSS  
Backend: Node.js, Express.js  
Database: MongoDB  
Authentication: JWT, bcrypt  
Version Control: Git & GitHub

## Key Features
- Role-based authentication (Customer / Artisan)
- Product listing & management
- Real-time chat for customization
- Secure payment options (UPI, Cards, Net Banking, COD)
- Order tracking & history
- Ratings & reviews
